# Sistema Industrial em PHP

## Descrição
Este é um sistema industrial desenvolvido exclusivamente em linguagem PHP. O objetivo é realizar o gerenciamento básico de processos e informações industriais de forma simples e funcional.

## Tecnologias utilizadas
- PHP

## Funcionalidades
- Cadastro de informações industriais
- Controle de processos
- Gerenciamento básico de dados
- Sistema simples para uso interno

## Como executar o projeto
1. Instale um servidor local como XAMPP ou WAMP
